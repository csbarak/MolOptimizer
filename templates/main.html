MolOptimizer

Welcome to Online Drug Discovery Simualtor developed at Akabavoy labs, Dept. of Chemistry, Ben-Gurion University of the Negev.
This website is free to use and provides features such as,

  • Alignment of Molecules
  • Extraction of Molecular Descriptors
  • Prediction of binding scores

Important Information regarding extracting of molecular descriptors

Extraction of molecular descriptors are from RDKit and Mordred libraries. The links to which are given below
  • Visit RDKit website
  • Visit Mordred Documentation
We do not claim authorship of the features extracted.

Molecules Alignment Section.

Please upload your files in SDF format (only). Alignment of molecules is done using RDKit Most Common Substructure (MCS Module). More details about the alignment of the files can be found in MolOptimizer paper.
The alignment of molecules is based on a reference molecule
Upload your reference molecule first and then upload your ligand library.

MANUAL MODE -- XGBoost

Binding Score Training Section -- XGBoost Regression(squared error loss function) with tree booster
Welcome to Binding Score prediction train section of Online Drug Discovery Web Tool.

Upload your csv files to train your model

Due to large number of features it is better to train the model on select features.
After uploading your csv file, please select the features on which you will like your model to be trained and click submit
Remember to put your binding scores under the column 'BOND'(all letters capitalized). The model will take this column as the one to predict

Tune your XGBoost Model

It is advisable to refer to XGBoost documentation to have a deeper understanding about the parameters.
Link to XGBoost Documentation

Please do be cautious while entering the values below.

Kindly make sure to enter only numbers without spacing before or after the numbers

Parameters for Tree Booster

Enter the learning rate which you desire, Range[0,1]  
Enter the max_depth value which you desire, Range[0,∞]

Enter the lambda value which you desire, [Increase in value will increase make model more conservative]
Enter the alpha value which you desire, [Increase in value will increase make model more conservative]

Enter drop rate value which you desire, Range [0.0,1.0]

LASSO REGRESSION

About this section
In this section Lasso Regression model can be trained. The optimum alpha value for Lasso Regression model will be automatically found in an iterative method through GridSearchCV.
Top 10 most influential features will be recommended

This section works as follows,

→ Optimum alpha value for lasso regression model is found

→ Model is re-trained with the optimum alpha value.Most influential features are detected from the model.

→ Option to re-train the model with the most influential features
